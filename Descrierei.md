##### Tipul aplicatiei

##### 

* site web al unei librarii

##### 

##### Scopul aplicatiei

##### 

\* gestionarea cartilor din librarie, a categoriilor si a editurilor de care o carte poate apartine;

\* realizarea functionalitatilor legate de cosul de cumparaturi (adaugarea unei carti, stergerea unei carti, modificarea cantitatii unui item din cos si finalizarea unei comeni).

##### 

##### Instrumente folosite

##### 

\* php, HTML, JavaScript, CSS, Bootstrap;

\* Apache Solr -> pentru a realiza cautarile complexe;

\* Form Submit -> pentru a permite utilizatorilor sa trimita un mail direct din aplicatie;

\* Youtube Data API -> pentru a afisa si accesa videoclipuri legate de o anumita carte;

\* Books API -> pentru determinarea rating-ului unei carti din libraraie.

##### 

##### Functionalitatile aplicatiei

##### 

###### \* pentru admin:

##### 

&#x20; \* inserarea, steargerea sau editarea de carti, categorii si edituri.



###### \* pentru client:

##### 

&#x20; \* vizualizarea paginii Home;

&#x20; \* vizualizarea tuturor cartilor sau a cartilor dintr-o anumita categorie;

&#x20; \* vizualizarea detaliilor unei carti si pdf-ul asociat ei;

&#x20; \* trimiterea unui mail folosind FormSubmit;

&#x20; \* efectuarea de cautari complexe asupra pdf-urilor asociate cartilor;

&#x20; \* cautarea unei carti dupa titlu;

&#x20; \* crearea unui cont, logarea si delogarea, editarea profilului;

&#x20; \* efectuarea de operatii asociate cosului de cumparaturi.

##### 

##### API-uri interne

##### 

\* pentru categorii -> endpoint-uri pentru operatiile de: inserare, editare, stergere si obtinerea tuturor categoriilor;

\* pentru edituri -> endpoint-uri pentru operatiile de: inserare, editare, stergere si obtinerea tuturor editurilor;

\* pentru contul de utilizator -> endpoint-uri pentru operatiile de: creare de cont, logare si editarea datelor unui cont.
